# AR Consultancy - UAE Business Setup Website

A modern, feature-rich React.js website for AR Consultancy, offering comprehensive UAE business setup services. This is a complete rebuild of the original website with advanced interactive features and professional design.

## Features

### 🎯 Advanced Functionality
- **Interactive Quote Calculator** - Real-time pricing based on selected services
- **Live Chat Widget** - 24/7 customer support integration
- **Service Package Comparison** - Side-by-side comparison of offerings
- **Testimonials Carousel** - Auto-rotating client success stories
- **FAQ Search** - Searchable frequently asked questions
- **Document Checklist** - Track required documents with progress bar
- **Contact Forms** - Multi-field form collection with validation
- **Newsletter Signup** - Email subscription management
- **Timeline Visualization** - 3-step business setup process
- **Responsive Design** - Mobile-first, works on all devices

### 🎨 Design & UX
- Professional blue and gold color scheme optimized for B2B services
- Smooth animations and transitions
- Accessible design with proper ARIA labels
- Optimized for SEO with meta tags and structured content
- Mobile-responsive layout using Tailwind CSS

### 📱 Components
- **Header** - Navigation with mobile menu
- **Hero** - Compelling value proposition with social proof
- **Features** - Key benefits of UAE business setup
- **Services** - 6 core service offerings
- **PackageComparison** - Three pricing tiers
- **QuoteCalculator** - Customizable pricing tool
- **Timeline** - Business setup process visualization
- **DocumentChecklist** - Interactive document tracker
- **Testimonials** - Client success stories carousel
- **FAQ** - Searchable Q&A section
- **ContactForm** - Lead capture form
- **Newsletter** - Email subscription
- **ChatWidget** - Support chat interface
- **Footer** - Company info and links

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **UI Components**: shadcn/ui
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Language**: TypeScript
- **State Management**: React Hooks

## Getting Started

### Installation

1. Clone or download the project
2. Install dependencies:
```bash
pnpm install
```

3. Run the development server:
```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

### Customization

#### Update Content
- Edit component files in `/components` folder
- Update text, prices, and service descriptions
- Add your company logo and branding

#### Modify Styling
- Update color scheme in `/app/globals.css`
- Adjust Tailwind classes in component files
- Customize fonts in `/app/layout.tsx`

#### Add Integrations
- Connect newsletter to email service (Mailchimp, ConvertKit, etc.)
- Integrate chat widget with support platform (Intercom, Zendesk, etc.)
- Add form submissions to CRM (HubSpot, Salesforce, etc.)
- Implement analytics (Google Analytics, Mixpanel)
